[![Build Status][travis-image]][travis-url]

> Yoctol specific configuration for PostCSS


## Install

```
$ npm install postcss-preset-yoctol
```


## Usage

`postcss.config.js`:

```js
module.exports = require('postcss-preset-yoctol');
```
